Most Recent: lolupOpen Source

Are the League of Legends servers online?

Website

lolup lets you know if the League of Legends servers are online. It will display a message as well as a green background if the game and store are online, a yellow background if the game is online but the store is offline, or a red background if the game is offline.

This project is a learning experience as I'm getting to know more about GitHub pages, Javascript, AngularJS, as well as working with REST endpoints. The controller uses the Riot API REST endpoints specific to the chosen region to determine the status of the servers. A specific region can be accessed by going to http://eperjesi.me/lolup/#/{region}, for example http://eperjesi.me/lolup/#/na. Source code can be found on GitHub.
