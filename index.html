Fayvrit

Find what others have been looking for!

Website

Fayvrit lets you browse Amazon products that others have posted to reddit. You get the option of scrolling through all of the most recent or you can specify a subreddit and it will narrow your search.

This project is a learning experience as I'm getting to work with frameworks I've never seen before. It is designed around AngularJS so the whole site is technically a single page application. Bootstrap allows the site to be responsive no matter which device you view it on. All of the data is three-way bound using Firebase and AngularJS, which allows it to load new data onto the page as soon as it hits the database. It also allows the narrowing of searches instantly and with no page refreshing. Lastly I got to work with Heroku for the first time as that's where this app is hosted.
